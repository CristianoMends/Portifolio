Sobre CSS3

O que é CSS

CSS é uma linguagem de estilo usada para definir a aparência e o layout de páginas da web escritas em HTML ou XHTML. Ele permite aos desenvolvedores separar a apresentação visual de uma página da sua estrutura de conteúdo, usando o conceito de "cascata" para determinar a prioridade de um estilo em relação a outro. Com o CSS, é possível criar uma ampla variedade de estilos e layouts, desde páginas simples até designs altamente complexos e sofisticados, sendo uma habilidade essencial para qualquer desenvolvedor web.

Propriedades de estilo para cada seletor, como background-color, color, padding, font-size, font-weight, margin, line-height, text-align e muito mais.

Comentários CSS (/* comentário */) para explicar o código e torná-lo mais legível.

É importante notar que o CSS deve ser vinculado ao HTML usando o elemento no da página ou incluído no próprio documento HTML usando o elemento style.

Transform

Matrix

O transform: matrix é uma propriedade CSS que permite aplicar uma transformação matricial a um elemento HTML.

A sintaxe do matrix é:
Transform: matrix(a,b,c,d,e,f);
estes são os significados de cada valor:
a e d: Escala horizontal e vertical, respectivamente.
b e c: Cisalhamento horizontal e vertical, respectivamente.
e e f: Translação horizontal e vertical, respectivamente.
