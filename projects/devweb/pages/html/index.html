O que é HTML?

HTML é a sigla para "HyperText Markup Language" (Linguagem de Marcação de Hipertexto, em português) e é a linguagem padrão usada para criar páginas da web. O HTML usa tags para estruturar o conteúdo de uma página da web, incluindo texto, imagens, links e outros elementos. Os navegadores da web interpretam o código HTML para exibir o conteúdo da página aos usuários.
